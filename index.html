Selected Commands

A small selection of frequently used commands to get Agents and members started. See Features for the full command list.

/submit

Open the guided incident submission form for users to file reports.

/agentos-verify <roblox-username>

Start Roblox OAuth verification to link Discord and Roblox accounts.

/background-check <user>

Quick summary of prior suspensions, arrests, and flags for a user.

/suspension-add <user> <duration>

Record a suspension with reason and duration (logged with author).
